# 소개

TimeBlock Planner는 **게이미피케이션과 AI 동반자 시스템을 결합한 지능형 로컬-퍼스트 타임블로킹 플래너**입니다.

## 왜 TimeBlock Planner인가?

단순한 일정 관리를 넘어, **타임블로킹(Time-blocking)** 방법론을 기반으로:

- 🎮 **게임화 요소(RPG)** - XP, 레벨, 보스 레이드로 동기부여
- 🤖 **AI 가상 동반자(Waifu Companion)** - 정서적 지지와 상호작용
- 🧠 **Gemini AI + RAG** - 내 데이터를 기억하는 맥락 인식 AI

이 모든 것을 결합하여 깊은 몰입과 생산성을 이끌어냅니다.

## 핵심 철학

### Local-First 아키텍처

네트워크 없이도 완전한 기능을 제공합니다.

```
┌─────────────────────────────────────────────────┐
│              사용자 인터페이스                    │
├─────────────────────────────────────────────────┤
│              Zustand 스토어                      │
├─────────────────────────────────────────────────┤
│              Repository 계층                     │
├──────────────────────┬──────────────────────────┤
│   IndexedDB (Dexie)  │    Firebase (Cloud)      │
│      [Primary]       │     [Backup/Sync]        │
└──────────────────────┴──────────────────────────┘
```

### 심리적 저항 관리

단순히 시간만 배분하는 것이 아니라, **작업에 대한 심리적 저항(Resistance)**을 관리합니다.

| 저항도 | 배율 | 설명 |
|:---|:---:|:---|
| 🟢 Low | 1.0x | 즐겁고 가벼운 작업 |
| 🟡 Medium | 1.3x | 평범한 업무 |
| 🔴 High | 1.6x | 미루고 싶은 부담스러운 작업 |

## 대상 사용자

- ADHD를 가진 분들
- 시간 관리에 어려움을 겪는 분들
- 생산성 앱에 쉽게 지치는 분들
- 게이미피케이션으로 동기부여가 필요한 분들
