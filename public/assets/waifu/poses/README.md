# 와이푸 포즈 이미지 폴더

이 폴더에 호감도별 와이푸 이미지를 추가하세요.

## 이미지 파일명 규칙

호감도 구간에 따라 다음 파일명을 사용해야 합니다:

| 호감도 범위 | 파일명 | 설명 |
|------------|--------|------|
| 0-10% | `very_low.png` | 매우 불만 (빨강) |
| 11-30% | `low.png` | 불만 (오렌지) |
| 31-50% | `medium.png` | 보통 (노랑) |
| 51-70% | `good.png` | 좋음 (초록) |
| 71-85% | `very_good.png` | 매우 좋음 (파랑) |
| 86-100% | `max.png` | 최고 (핑크) |

## 이미지 요구사항

- **파일 형식**: PNG 또는 JPG
- **권장 크기**: 세로 이미지 (예: 400x600px, 500x750px)
- **배경**: 투명 배경 PNG 권장
- **파일 크기**: 2MB 이하 권장

## 사용 예시

```
public/assets/waifu/poses/
├── very_low.png    # 호감도 0-10% 이미지
├── low.png         # 호감도 11-30% 이미지
├── medium.png      # 호감도 31-50% 이미지
├── good.png        # 호감도 51-70% 이미지
├── very_good.png   # 호감도 71-85% 이미지
└── max.png         # 호감도 86-100% 이미지
```

## 작동 방식

1. WaifuPanel 컴포넌트는 현재 호감도를 확인합니다.
2. `waifuImageUtils.ts`의 `getWaifuImagePath()` 함수가 호감도에 맞는 이미지를 선택합니다.
3. 해당 이미지가 없으면 `/assets/waifu/default.png`를 사용합니다.
4. default.png도 없으면 플레이스홀더를 표시합니다.

## 팁

- 각 호감도 구간별로 캐릭터의 표정이나 포즈를 다르게 그리면 더 생동감 있는 경험을 제공할 수 있습니다.
- 호감도가 높을수록 밝고 긍정적인 이미지를 사용하세요.
- 투명 배경 PNG를 사용하면 UI와 더 자연스럽게 어울립니다.
